Основные настройки

Основные настройки программы доступны через меню Инструменты - Настройки на вкладке Основные.
Параметры программы разбиты на следующие группы:
Database
Local Database Location - Укажите или выберите в диалоговом окне полный путь к директории, в которой находится база данных. По умолчанию база находится в директории приложения в поддиректории db.
Remote Database URL - Укажите полный адрес удаленной базы данных. По умолчанию используется адрес, указывающий на локальную базу jdbc:hsqldb:hsql://127.0.0.1/jukes. Примечание: Этот адрес используется также генератором отчетов для доступа к БД.

Display
Disc Display Format - Укажите формат отображения названия альбома в заголовке окна альбома. Можно использовать следующие маски: %b - битрейт, %y - год, %r - время звучания, %d - название альбома. Например, [%y] %d (%r) %b kbps.
Track Display Format - Укажите формат отображения названия трека в заголовке окна трека. Можно использовать следующие маски: %b - битрейт, %n - номер трека, %r - время звучания, %t - название трека. Например, %n. %t (%r).
New File Marker (in days) - Укажите количество дней, в течение которых альбом будет считаться Новым (Недавно добавленным). По умолчанию используется значение 30. Новые альбомы выделены в дереве коллекции жирным шрифтом.
Catalog Scroll Units - Укажите скорость прокрутки каталога мышью. По умолчанию используется значение 5.
Audit Information - Этот флаг определяет, будут ли на экранах программы отображаться данные, предназначенные для контроля за работой с базой данных - дата и время создания и изменения записи, а также код пользователя, создавшего или изменившего запись. По умолчанию этот флаг включен.
Cover Size Small (in pixels) - Укажите малый размер в пикселях, до которого будет изменено изображение обложки альбома при отображении в списке альбомов в окне исполнителя, в окне трека и других окнах, в которых используется уменьшенное изображение. По умолчанию используется значение 91.
Cover Size Large (in pixels) - Укажите большой размер в пикселях, до которого будет изменено изображение обложки альбома при отображении в окне альбома. По умолчанию используется значение 200.
Row Color Even - Выберите цвет четной строки в таблицах, таких как таблица результатов поиска треков.
Row Color Odd - Выберите цвет нечетной строки в таблицах, таких как таблица результатов поиска треков.
Show Main Tree by Default - Этот флаг определяет, всегда ли отображается основное дерево коллекции. Если флаг выключен, то дерево отображает только альбомы, отобранные с помощью фильтра. Этот режим может быть полезен при большом объеме коллекции для ускорения работы программы. По умолчанию это флаг включен.
Show Empty Nodes in the Tree - Этот флаг определяет, будут ли отображаться в дереве узлы, не содержащие вложений (потомков). Например, исполнители, для которых в базе данных нет ни одного альбома. По умолчанию это флаг выключен.

Files and Directories
Music Directory - Укажите или выберите в диалоговом окне полный путь к директории, в которой программа будет по умолчанию начинать поиск аудио-файлов.
File Format Music - Укажите формат переименования музыкальных файлов. Можно использовать следующие маски: %n - номер трека, %t - название трека, %a - исполнитель, %d - название альбома. Например, %n - %t.
File Format Image - Укажите, формат переименования файлов изображений. Можно использовать следующие маски: %a - исполнитель, %d - название альбома. Например, %a - %y - %d. Можно также указать постоянное имя, например folder.jpg.
Default Playlist Type - Выберите тип плейлиста, используемый по умолчанию: XSPF или M3U.
Backup File - Укажите или выберите в диалоговом окне полный путь к директории, в которой будет создан резервный архив базы данных. По умолчанию файл архива будет называться backup и будет находиться в поддиректории backup директории приложения.
Logfile Level - Выберите уровень сообщений, записываемых в файл протокола. Уровень сообщений может быть: Debug, Info, Warning, Error. Файл протокола находится в поддиректории LOG директории приложения.
Copy Images - Этот флаг определяет, будут ли файлы изображений обложек альбомов, найденные в директориях альбомов, копироваться в указанную пользователем директорию, или нет. Путь к директории определяется параметром Images Directory. Этот режим работы может быть полезен, если айдио-файлы хранятся на внешних съемных устройствах, и обеспечивает доступность изображений обложек при просмотре содержимого коллекции. Формат имен файлов изображений при копировании определяется параметром File Format Image. Размер копируемых файлов определяется параметром Cover Size Large (in pixels). По умолчанию этот флаг выключен.
Images Directory - Укажите или выберите в диалоговом окне полный путь к директории, в которой находятся копии файлов изображений обложек. Формат имен файлов изображений определяется параметром File Format Image.

Player
Spectrum Analyzer Mode - Укажите режим анализатора спектра: Off, Analyzer, или Scope. В текущей версии анализатор спектра не используется. Параметр зарезервирован для использования в будущем.
Buffer Size (in bytes) - Укажите размер буфера аудио-плеера. При возникновении пробелов в звучании рекомендуется увеличить это число. По умолчанию указано значение 512 000.
Fade In On Play - Этот флаг определяет, будет ли использоваться эффект постепенного услинения звука при старте воспроизведения. По умолчанию этот флаг включен.
Fade Out On Change - Этот флаг определяет, будет ли использоваться эффект постепенного затухания звука при смене трека. По умолчанию этот флаг включен.
Fade Out On Pause - Этот флаг определяет, будет ли использоваться эффект постепенного затухания звука при паузе трека. По умолчанию этот флаг включен.
Fade Out On Stop - Этот флаг определяет, будет ли использоваться эффект постепенного затухания звука при остановке трека. По умолчанию этот флаг включен.

Tags
Use CD Number - Этот флаг определяет, будет ли при формировании названия альбома на основе тегов аудио-файлов использоваться тег "Disc Number". Если этот флаг включен, то к названию альбома будет добавлено Disc "Disc Number" of "Total Discs". Например, Disc 1 of 2. По умолчанию этот флаг выключен.
Album Artist Tag - Укажите название тега, который будет использоваться для определения поля Album Artist. Поле Album Artist используется для определения исполнителя альбома на основании тегов. По умолчанию указано "ALBUM ARTIST".
Update Audio Tags in Files - Этот флаг определяет, будут ли при внесении изменений в базу данных соответствующие изменения записаны в теги аудио-файлов. По умолчанию этот флаг включен.
