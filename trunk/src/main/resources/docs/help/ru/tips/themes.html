Если выбран внешний вид Plastic , то в меню Настройки на вкладке Стиль можно выбрать нужную цветовую схему.

В названии большинства цветовых схем указан цвет цвет выделенно элемента.

Цвета фона используются следующие: Desert - темносерый и хорошо подходит для Windows ME/2000,
Sky - светлосерый и подходит для большинства платформ,
Experience - оптимизирован для Windows XP.
