Вид окон программы можно изменить с помощью пункта меню Настройки.

Можно выбрать вид, предлагаемый по умолчанию для Вашей платформы, или платформо-независимый вид JGoodies Plastic.

Вид Plastic позволяет выбрать одну из нескольких цветовых схем, одна из которых - Experience - оптимизирована для Windows XP.
