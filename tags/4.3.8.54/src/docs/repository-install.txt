Jukes Maven Repo Instructions
------------------------------

1. Copy the entire jukes-dependencies.zip file into your C:\Users\USER\.m2\repository folder.

Or....

2. Do a install-file on each of the files in that zip file with the following commands.

mvn install:install-file -DgroupId=com.jcraft -DartifactId=jogg -Dversion=0.0.7 -Dpackaging=jar -Dfile=jogg-0.0.7.jar -DgeneratePom=true
