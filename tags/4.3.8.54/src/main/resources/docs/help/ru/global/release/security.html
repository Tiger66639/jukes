Безопасность

Код с цифровой подписью

Для обеспечения проверки целостности и идентичности код программы подписан цифровой подписью с помощью сертификата публичного ключа следующего владельца: "CN=Emil Lefkof, OU=melloware.com, O=Emil Lefkof, L=Media, ST=PA, C=US", что подтверждается службой сертификации Thawte.

Если для запуска приложения Вы используете технологию, совместимую с Java Web Start, то код будет автоматически проверен на идентичность при запуске. Если Вы располагаете Java Development Kit (JDK), то можете проверить целостность данных с помощью jarsigner.

Связь по Интернет

Программа не устанавливает и не требует для работы связи по Интернет. Создаваемый отчет об ошибках не будет отправлен автоматически. Его следует отправить с помощью любой стандартной программы для электронной почты. Для облегчения отправки отчета об ошибке программа предоставляет опцию "Copy-to-Clipboard" , которая позволяет поместить содержание отчета об ошибке в буфер.

Защита персональных данных

При появлении ошибок в ходе работы программы в поддиректории log создается файл протокола console.log. Этот файл содержит информацию об операционной системе, ее версии и конфигурации компьютера. Эта информация может быть полезной для обнаружения ошибок. Перед отсылкой сообщения об ошибке Вы можете удалить ее для обеспечения защиты персональных данных.
