Did you know...

If you use the tool with the Plastic look, you can choose a color theme in the Preferences.

Most themes have a prefix for the background color: Desert is brown and works well on Windows ME/2000, Sky is light gray and works well on most platforms, and Experience has been optimized for Windows XP.

The theme's suffix is for the foreground color and saturation, where Bluer is more saturated than Blue.
