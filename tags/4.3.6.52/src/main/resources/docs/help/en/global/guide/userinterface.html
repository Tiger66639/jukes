User Interface

Pull-down Menu
The pull-down menu provides you with a complete list of actions available in the application.

Toolbar
The tool bar gives you quick access to the most important actions.

Navigation tree
The navigation tree in the left side of the application window shows artists and albums included into music collection in a clear graphical form.

View and Edit forms
View and Edit forms allows to view all detailed information about Artists, Albums and tracks including technical audit information, enter or correct this information and search for additional information in Internet.

Filter panel
Filter panel in the left-bottom part of the application window allows to restrict Navigation tree by Artist name, Album title, release year or genre. Also you can select only recently added albums.

Popup Menus
A popup menu is available for each selected item - Atrist, Album, Track or cover images - and includes most frequently used actions such as adding to playlist or playing.
